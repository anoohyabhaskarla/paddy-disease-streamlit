# Paddy Disease Detection Dashboard

This Streamlit app allows you to classify paddy diseases from leaf images and weather data.

## How to Run

1. Install requirements:

```
pip install -r requirements.txt
```

2. Make sure your trained model file is in the same folder (e.g. `paddy_model_fusion_final.h5`)

3. Launch the app:

```
streamlit run app.py
```

Then open `http://localhost:8501` in your browser.
